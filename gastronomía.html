Inicio Gobierno y Sociedad Cultura Gastronomía Turismo

Kaq Ik

Fue declarado patrimonio cultural intangible en el año 2007. Este delicioso plato consiste en un pavo hervido con ajo, hierbabuena y cilantro. Este se puede juntar con arroz o con unos tamales.

Pepián

Este plato se puede hacer con carne de res, costilla de cerdo, pollo. Este plato surgió de una mezcla de culturas entre la maya y la española. Existen istintos tipos de este plato, por ejemplo, el pepián negro, el rojo, o incluso el de tres carnes. Sin duda este plato es ingenioso y variado.

Fiambre

El Fiambre es una mezcla exquisita entre embutidos, verduras, quesos o carne. Usualmente las verduras base de este plato son los pimientos, ajo, tomillo. Para las carnes hay una alta variedad de ingredientes. Por ejemplo, el salchichón, lengua de vaca, pollo, atún, salami, hasta camarones.
